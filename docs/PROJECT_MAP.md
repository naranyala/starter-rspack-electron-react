# Project Map

This map points to the places you are most likely to touch when making changes.

## Top Level

- `main.ts` Electron main process entry (source of truth).
- `preload.ts` Preload entry used at runtime (source of truth).
- `main.cjs` and `preload.js` Runtime artifacts generated by `tsc` and `postbuild`.
- `rspack.config.ts` Rspack config for renderer bundling.
- `package.json` Scripts, deps, and electron-builder config.
- `scripts/` Build and dev scripts (TypeScript sources plus compiled JS).
- `src/` Renderer UI, shared utils, and main-process code used by handlers.
- `dist/` Renderer build output (generated).
- `dist-ts/` TypeScript compiler output (generated).

## Renderer (UI)

- `src/index.tsx` React entry point.
- `src/App.tsx` Main UI shell and feature cards.
- `src/renderer/components/` Sidebar and UI components.
- `src/renderer/features/` WinBox window creators (one per feature).
- `src/renderer/utils/winbox-utils.ts` WinBox window creation and layout logic.
- `src/renderer/lib/` Frontend utilities and helpers.
- `src/renderer/styles/goober.ts` Styling tokens and helpers.

## Main Process

- `main.ts` Electron lifecycle and BrowserWindow creation.
- `src/main/handlers/` IPC handler registration and demo content.
- `src/main/lib/` Helper libraries (ipc, filesystem, security, etc.).
- `src/main/preload.ts` Example preload bridge (not referenced by `main.ts`).

## Shared

- `src/shared/` Menu data, window content generator, and shared utils.
- `src/types/` Type declarations.
- `src/assets/` Icons and images copied to `dist/`.
